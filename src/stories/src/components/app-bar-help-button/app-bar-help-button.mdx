## Usage

For convenience Forge provides the `<forge-app-bar-help-button>` component. This component uses an icon-button
(with a predefined icon), and is wrapped in a Forge [menu component](?path=/story/components-menu--default). The power of this component is that it will ensure a
consistent experience for users across all applications that consume it.

> It is expected that you place this component in the `end` slot of the app-bar.

</PageSection>

## Events

| Name                      | Description
| :-------------------------| :-------------------
| `forge-menu-select`       | Emits when selecting an item from the menu. This is the same event as the standard `<forge-menu>` component. The event type is `CustomEvent` and contains an object in the `detail` property that has an `index` and `value` property in it.

## Types

### IMenuOption

```ts
interface IMenuOption<T = any> {
  value: T;
  label: string;
  disabled?: boolean;
  divider?: boolean;
  optionClass?: string | string[];
  leadingIcon?: string;
  leadingIconClass?: string;
  leadingIconType?: ListDropdownIconType;
  trailingIcon?: string;
  trailingIconClass?: string;
  trailingIconType?: ListDropdownIconType;
  leadingBuilder?: () => HTMLElement;
  trailingBuilder?: () => HTMLElement;
  options?: Array<IListDropdownOption | IListDropdownOptionGroup>;
  elementAttributes?: Map<string, string>;
  icon?: string;
  selected?: boolean;
}
```

### ListDropdownIconType

```ts
type ListDropdownIconType = 'font' | 'component';
```
