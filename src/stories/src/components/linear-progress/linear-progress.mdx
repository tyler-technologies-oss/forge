# Linear progress

Linear progress indicators display progress by animating along a linear track in a horizontal direction.

Progress indicators inform users about the status of ongoing processes.
 - Determinate indicators display how long a process will take.
 - Indeterminate indicators express an unspecified amount of wait time.

</PageSection>

<PageSection>

## Example

### Indeterminate

Use this variant when the progress completion percentage is unknown.

<LiveDemo fullWidth codeHtml={LinearProgressIndeterminateCodeHtml()}>
  <LinearProgressDemo />
</LiveDemo>

### Determinate

Use this variant when you know exactly how much progress has been made out of a total number of items.

<LiveDemo fullWidth codeHtml={LinearProgressDeterminateCodeHtml()}>
  <LinearProgressDemo determinate={false} />
</LiveDemo>

### Buffer

This variant is similar to determinate, except it allows you to show that the total amount of items is still being determined.

<LiveDemo fullWidth={true} codeHtml={LinearProgressBufferCodeHtml()}>
  <LinearProgressDemo progress={0.33} buffer={0.66} />
</LiveDemo>

## Properties/Attributes

<PropertyDef name="determinate" type="boolean" defaultValue="false">

Controls whether the component is using the determinate (`true`) or indeterminate (`false`) variant.

</PropertyDef>

<PropertyDef name="progress" type="number" defaultValue="0">

The progress amount (between `0` and `1`) of the indicator. Only valid in determinate state.

</PropertyDef>

<PropertyDef name="buffer" type="number" defaultValue="1">

The buffer amount (between `0` and `1`) of the indicator. Only valid in determinate state.

</PropertyDef>

<PropertyDef name="data-aria-label" type="string" defaultValue="''" prop={false}>

Sets the `aria-label` attribute on the internal `progressbar` element.

> **Note:** You should **always** set this attribute for compliance with accessibility requirements.

</PropertyDef>

</PageSection>

## CSS custom properties

| Name                                                      | Description
| :---------------------------------------------------------| :----------------
| `--forge-linear-progress-indicator-color`                 | The color of the progress indicator.
| `--forge-linear-progress-indicator-background-color`      | The background color of the progress indicator.
| `--forge-linear-progress-height`                          | The height of the progress indicator.

</PageSection>

<PageSection>

## Accessibility

- Ensure that the progress indicator has a label that describes what the progress indicator represents.
- Ensure the progress indicator provides a role attribute with value "alert", and an `aria-busy` attribute with the value `"true"` while the progress indicator is active.

</PageSection>
