# Card

The card component is intended to be used as a layout design element to help group related content together into sections.

This a simple component that doesn't provide any functionality, but brings consistency through common default values for `padding`, `box-shadow`, `border` and some
structure within its template to help support fixed header and footer while allowing for a scrollable body. You are free to place any content you would like into this
component using its default slot.

See the [CSS custom properties](#css-custom-properties) section below for information on internal styles that you can override.

## Properties/Attributes

<PropertyDef name="outlined" type="boolean" defaultValue="null">

Sets the card to use a flat appearance with a standard `border` instead of the raised `box-shadow`.

> Never use a standard raised card on another raised surface. Use the `outlined` type if you need to place cards on other cards for example.
  
</PropertyDef>

<PageSection>

## CSS custom properties

| Name                              | Description
| :---------------------------------| :----------------
| `--mdc-theme-surface`             | Controls the surface (background) color of the card.
| `--forge-card-padding`              | Controls the `padding` of the container element.
| `--forge-card-overflow`             | Controls the `overflow` style of the card element.
| `--forge-card-height`               | Sets the `height` of the card element.
| `--forge-card-width`                | Sets the `width` of the card element.
| `--forge-theme-border-color`        | Sets the border color of the outlined card element.

</PageSection>
