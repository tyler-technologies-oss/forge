# App Bar Notifications

For convenience Forge provides the `<forge-app-bar-notification-button>` component. This component uses an icon-button
with a predefined icon. The power of this component is that it will ensure a consistent experience for users
across all applications that consume it.

If your application provides system notifications, use this component to show those notifications to the user.

> It is expected that you place this component in the `emnd` slot of the app-bar.

</PageSection>

## Properties/Attributes

<PropertyDef name="count" type="string | number | null | undefined" defaultValue="0">

  Gets/sets the count value of the notification button.

</PropertyDef>

<PropertyDef name="dot" type="boolean" defaultValue="false">

  Gets/sets whether the badge shows dots.

</PropertyDef>

<PropertyDef name="showBadge" type="boolean" defaultValue="false">

  Gets/sets whether the notification button shows the badge.

</PropertyDef>

<PropertyDef name="theme" type="string" defaultValue="'secondary'">

  Gets/sets the theme of the badge.

</PropertyDef>

<PropertyDef name="icon" type="string" defaultValue="'notifications'">

Gets/sets the icon of the notification button

</PropertyDef>
