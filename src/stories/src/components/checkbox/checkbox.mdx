# Checkbox

This component is purely used for styling purposes. The native `<input type="checkbox">` element is left exposed, and you are to interact with that element as
you would any other native `<input type="checkbox">`.

> If you are using a framework such as Angular, React, Vue... etc. be sure that all bindings are placed on the `<input>` element. Do NOT
> place any bindings on the `<forge-checkbox>` element as those will not be propagated through to the `<input>`.
