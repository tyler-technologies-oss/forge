# Label value

Use the label-value component when you have data that needs to be displayed, but cannot be edited. This is generally ideal for a read-only screen, or 
toggling between viewing and editing data values on a form.

This component is simple, but it provides consistency and ease of use through slotted content to remove the need for developers to worry about CSS styles. It also
matches the dimensions of other field field elements such as text-field and select to enable them to live next to each other on a page without adjustments.

## Properties/Attributes 

<PropertyDef name="empty" type="boolean" defaultValue="false">

Controls the typography of the value text when a no value can be displayed.

You will typically set the text to *n/a* when using this property.
  
</PropertyDef>

<PropertyDef name="density" type="LabelValueDensityType" defaultValue="'default'">

Gets/sets the density state
  
</PropertyDef>

<PropertyDef name="align" type="LabelValueAlignment" defaultValue="&quot;left&quot;">

Controls the cross-axis alignment of the text. Valid values: `left` (default), `center`, `right`.
Valid values: `left`, `center`, `right`.
  
</PropertyDef>

<PropertyDef name="ellipsis" type="boolean" defaultValue="false">

Gets/sets whether label text will wrap or use ellipsis when overflowing.
  
</PropertyDef>

<PropertyDef name="single-line" type="boolean" prop={false} defaultValue="undefined">

Gets/sets whether label and value will be displayed on a single row.
  
</PropertyDef>

<PageSection>

## Slots

| Name               | Description
| :------------------| :------------------
| `label`            | Places content in the `label` slot within the component. Proper label typography is applied for you.
| `value`            | Places content in the `value` slot within the component. Proper value typography is applied for you.
| `icon`             | Places top-aligned content to the left of the `label` and `value` slots.

</PageSection>

<PageSection>

## CSS parts

| Name                      | Description
| :-------------------------| :------------------
| `root`                    | The root element.
| `icon-container`          | The icon container element.
| `container`               | The layout container element.
| `label-container`         | The container element around the `label` slot.
| `value-container`         | The container element around the `value` slot.

</PageSection>
