## Usage

The button component is nothing more than a wrapper around the native `<button>` element. It provides default styles, as well as manages the
ripple effect on the button. Along with that, you can adjust the button via the API provided through the component.

</PageSection>

## Properties/Attributes

<PropertyDef name="type" type="string" defaultValue="undefined">

Gets/sets the button type. Valid values: `raised`, `unelevated`, `outlined`, `dense`.
  
</PropertyDef>

> To use a dense variant, you can append the `-dense` suffix to any of the existing types. Ex. `outlined-dense`.

</PageSection>

<PageSection>

## CSS custom properties

| Name                                    | Description
| :---------------------------------------| :----------------
| `--mdc-theme-primary`                   | Sets the background color
| `--mdc-theme-on-primary`                | Sets the font color.
| `--forge-theme-border-color`            | Sets the border color.
| `--mdc-theme-text-disabled-on-light`    | Sets the disabled color.

> **Note:** To make a button fill 100% of its available width, put the `<forge-button>` inside a block element and set
> the width of both the `<forge-button>` component and its embedded `<button>` element to 100%.

</PageSection>
