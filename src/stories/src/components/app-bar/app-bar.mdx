## Usage

The app-bar should be used as the header for every page in your application to ensure consistency across our
Tyler products and applications. It's the one constant across all applications that helps ensure our applications
are perceived as being connected and familiar.

The app-bar itself is nothing more than a container for other add-on components, but it does provide named slots
and a common location for branding logo and title.

## Properties/Attributes

<PropertyDef name="titleText" type="string" defaultValue="undefined">

  Gets/sets the title text.

</PropertyDef>

<PropertyDef name="fixed" type="boolean" defaultValue="false">

  Toggles the fixed variant. This variant will add a transition to the `box-shadow` style to blend with scroll events when using a sticky scrollable app-bar.

</PropertyDef>

<PropertyDef name="raised" type="boolean" defaultValue="true">

  Toggles the box-shadow (elevation) that is applied to the bottom of the app-bar.

</PropertyDef>

<PropertyDef name="logo" type="boolean"  defaultValue="true">

  Gets/sets the visibility of the logo.

</PropertyDef>

<PropertyDef name="theme" type="string | null" prop={false}  defaultValue="primary">

  Controls the theme color applied to the app-bar background. Primary is used for backoffice apps, white is used for citizen apps.
  
  Valid values: `primary` (default), `white`

</PropertyDef>

## Slots

| Name                 | Description
| :--------------------| :-----------------
| `start`              | Places content at the far left of the app-bar. Generally a menu button is used here.
| `logo`               | Places content in the logo slot to the left of the title text. Use this slot to override the default Tyler logo.
| `title`              | Overwrites the default title element with custom content.
| `center`             | Places content at the center of the app-bar.
| `end`                | Places content to the far right of the app-bar.
| `bottom`             | Adds a second row to the app-bar. This allows for placing content such as tabs for example.

</PageSection>

## CSS custom properties

| Name                                                | Description
| :---------------------------------------------------| :----------------
| `--forge-app-bar-theme-on-background`                 | Sets the font color.
| `--forge-app-bar-theme-background`                    | Sets the background color.
| `--forge-app-bar-theme-secondary-on-background`       | Sets the `--mdc-theme-text-secondary-on-background`.
| `--forge-app-bar-theme-secondary-on-background`       | Sets the `--mdc-theme-text-secondary-on-light`.
| `--forge-app-bar-theme-disabled-on-background`        | Sets the `--mdc-theme-text-disabled-on-background`.
| `--forge-app-bar-theme-disabled-on-light`             | Sets the `--mdc-theme-text-disabled-on-light`.
