# Stack

The stack component manages the spacing of immediate children along the vertical or horizontal axis with optional wrap, gap, alignment, and direction properties.

This component is useful in situations where you need to place elements horizontally or vertically next to each other with appropriate spacing between them. One example would be placing a save and cancel button side by side in the footer of a card. It's also useful if you have a vertical arrangement of elements and you want perfect spacing between them (known as vertical rhythm in the design world). This component allows you to handle those scenarios directly from the HTML reducing the need to add additional CSS classes or inline styles.

> **Note**: The stack component should not be used for a complex arrangement of child elements, and it's not intended to be used in full page layouts

## Properties/Attributes

<PropertyDef name="inline" type="boolean" defaultValue="false">

Adding the inline attribute will change the arrangement of child elements from vertical to horizontal within the stack. 

</PropertyDef>

<PropertyDef name="wrap" type="boolean" defaultValue="false">

Wrap is used to wrap child elements to a new row when there's not enough space. This only works when inline is set to true.

</PropertyDef>

<PropertyDef name="stretch" type="boolean" defaultValue="false">

Stretch allows each child element to take up as much width as possible until the parent container is filled.  You can override this on each individual child element by changing its `--forge-stack-stretch` CSS custom property. 

</PropertyDef>

<PropertyDef name="gap" type="string" defaultValue="16">

Controls the amount of space between child elements within a stack.

</PropertyDef>

<PropertyDef name="alignment" type="string" defaultValue="&quot;start&quot;">

  Align child elements within a stack. Valid values: `start` (default), `center`, `end`.

</PropertyDef>

## CSS custom properties

| Name                                          | Description
| :---------------------------------------------| :----------------
| `--forge-stack-stretch`                         | Controls the flex grow factor of a child element within the stack.

> **Tip:** Check out the MDN web docs on flex-grow to learn more: [MDN: flex-grow](https://developer.mozilla.org/en-US/docs/Web/CSS/flex-grow).
