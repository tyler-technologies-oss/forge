# Circular progress

Circular progress indicators display progress by animating along a circular track in a clockwise direction.

Progress indicators inform users about the status of ongoing processes.

  - Determinate indicators display how long a process will take.
  - Indeterminate indicators express an unspecified amount of wait time.

## Properties/Attributes

<PropertyDef name="determinate" type="boolean" defaultValue="false">

Gets/sets the determinate state.

**false:** indeterminate (continuous) animation.

**true:** allows for setting specific progress percentage amount to be controlled manually.

</PropertyDef>

<PropertyDef name="progress" type="number" defaultValue="0">

Gets/sets the progress percentage. Valid values are between `0` and `1`.

</PropertyDef>

## CSS custom properties

| Name                                                    | Description
| :-------------------------------------------------------| :----------
| `--forge-circular-progress-size`                        | Sets the indicator diameter (heigth and width). Default is `48px`.
| `--forge-circular-progress-padding`                     | Sets the padding between the container and the indicator track. Default is `0`.
| `--forge-circular-progress-indicator-width`             | Sets the indicator track width.
| `--forge-circular-progress-indicator-color`             | Sets the indicator theme color. Inherits from the tertiary theme by default.
| `--forge-circular-progress-indicator-background-color`  | Sets the indicator background color. Inherits from the text-disabled-on-background theme by default.

</PageSection>

<PageSection>

## Accessibility

* Ensure the progress indicator provides a role attribute with value "alert", and an `aria-busy="true"` attribute while the progress indicator is active.
