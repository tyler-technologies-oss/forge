# Circular progress

The circular progress is a configurable component that can be used for both determinate and indeterminate busy states. It renders as an svg,
and the properties can be controlled dynamically through the component and CSS custom properties.

## Properties/Attributes

<PropertyDef name="open" type="boolean" defaultValue="true">

Controls the visibility of the circular progress.

</PropertyDef>

<PropertyDef name="determinate" type="boolean" defaultValue="false">

Gets/sets determinate.

**false:** continuous animation.

**true:** allows for setting specific progress percentage amount to be controlled manually.

</PropertyDef>

<PropertyDef name="progress" type="number" defaultValue="0">

Gets/sets the progress percentage. Valid values are between 0 and 1.

</PropertyDef>

</PageSection>

## CSS parts

| Name                                            | Description
| :-----------------------------------------------| :----------
| `root`                                          | The root element.
| `determinate-container`                         | The determinate container element.
| `determinate-svg`                               | The determinate `<svg>` element.
| `determinate-svg-circle-track`                  | The determinate track SVG `<circle>` element.
| `determinate-svg-circle`                        | The determinate circle SVG `<circle>` element.
| `indeterminate-container`                       | The indeterminate container.
| `spinner-container`                             | The spinner container.
| `spinner-circle-left`                           | The left circle container.
| `spinner-circle-left-svg`                       | The left circle container `<svg>` element.
| `spinner-circle-left-svg-circle`                | The left circle SVG `<circle>` element.
| `spinner-gap`                                   | The spinner gap container element.
| `spinner-gap-svg`                               | The spinner gap `<svg>` element.
| `spinner-gap-circle`                            | The spinner gap SVG `<circle>` element.
| `spinner-circle-right`                          | The right circle container.
| `spinner-circle-right-svg`                      | The right circle container `<svg>` element.
| `spinner-circle-right-svg-circle`               | The right circle SVG `<circle>` element.

</PageSection>

<PageSection>

## CSS custom properties

| Name                                            | Description
| :-----------------------------------------------| :----------
| `--forge-theme-tertiary`                          | Sets the stroke color.
| `--forge-circular-progress-track-color`           | Sets the track (background) color.
| `--forge-circular-progress-size`                  | Sets the diameter of the circle. Default is `72px`.
| `--forge-circular-progress-stroke-width`          | Sets the stroke width of the outline. Default is `4`.

</PageSection>
