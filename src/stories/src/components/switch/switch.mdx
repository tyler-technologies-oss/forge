# Switch

Switches toggle the state of a single setting on or off. 

Use switches to:

  - Toggle a single item on or off, on mobile and tablet
  - Immediately activate or deactivate something

</PageSection>

<PageSection>

## Properties/Attributes

<PropertyDef name="dense" type="boolean" defaultValue="false">

  Gets/sets the dense state.

</PropertyDef>

<PropertyDef name="disabled" type="boolean" defaultValue="false">

  Gets/sets the disables state.

</PropertyDef>

<PropertyDef name="selected" type="boolean" defaultValue="false">

  Gets/sets the selected state.

</PropertyDef>

<PropertyDef name="labelPosition" type="SwitchLabelPosition" defaultValue="'end'">

  Gets/sets the logical side of the switch that the label should be positioned on.

</PropertyDef>

<PropertyDef name="button-aria-label" prop={false} type="string" defaultValue="undefined">

  Sets the `aria-label` on the internal `<button>` element.

  Only apply this attribute if you aren't using a label element, or need to specify an alternate label.

</PropertyDef>

</PageSection>

<PageSection>

## Events

| Name                              | Description
| :---------------------------------| :-----------------
| `forge-switch-select`               | Emits when the switch has changed. 

</PageSection>

<PageSection>

## Keyboard shortcuts

| Name                                | Description
| :-----------------------------------| :----------------
| `space / enter                      | Toggles the switch on / off.

</PageSection>

<PageSection>

## Slots

| Name               | Description
| :------------------| :--------------------
| `default`          | The default (unnamed) slot where label text will be placed.

</PageSection>
