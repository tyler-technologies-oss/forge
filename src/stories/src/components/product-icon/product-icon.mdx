# Product icon

The product icon component facilitates the creation of a long shadow effect on icons that are used primarily to represent application/product icons.
It takes in a color from the material color palette, and automatically adjusts the color of the shadow based on the palette being used.

Among other things, you can configure the size, shadow effect, and shape of the component if desired. The only requirement is that you provide
an icon or text. The icon must be a font-based icon. SVG icons (which includes the `<forge-icon>` component) are not supported by this component.

> **Note:** this component will also accept plain text content in place of a font-based icon.

## Properties/Attributes

<PropertyDef name="color" type="string" defaultValue="indigo-500">

The color of the circle container that holds the icon. Accepts colors from the material palette (ie. `indigo-500`, `deepPurple-500`).

</PropertyDef>

<PropertyDef name="size" type="number" defaultValue="64">

Sets the size of the container (in pixels).

A property/attribute is used instead of CSS due to the fact that the component needs to regenerate its shadow when the size changes.

</PropertyDef>

<PropertyDef name="shadow" type="boolean" defaultValue="true">

Controls the visibility of the shadow.

</PropertyDef>

<PropertyDef name="iterations" type="number" defaultValue="32">

This is used to generate the length of the shadow. Set this to exactly half of the size attribute to achieve the correct effect.

</PropertyDef>

</PageSection>

<PageSection>

## Slots

| Name               | Description
| :------------------| :----------------
| `default`          | The default (unnamed) slot is where you'll place your icon element.
| `text`             | If using text content without an icon, place your content in this slot so that the component can properly switch modes.

</PageSection>

<PageSection>

## CSS parts

| Name                          | Description
| :-----------------------------| :----------------
| `root`                        | The root element.
| `container`                   | The layout container element.
| `icon-container`              | The container around the `default` slot for use with icons.
| `text-container`              | The container around the `text` slot for use with plain text.

</PageSection>

<PageSection>

## CSS custom properties

| Name                                  | Description
| :-------------------------------------| :----------------
| `--forge-product-icon-border-radius`    | Controls the border radius of the container. Use this is you don't want a completely circular icon.
| `--forge-product-icon-font-size`        | Controls the font-size of the icon.

</PageSection>
