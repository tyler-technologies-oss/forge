# Page state

The page-state component is a layout design component that was created to ensure consistency, and provide structure for pages that need to
display a state such as empty, error, 404... etc. This will allow us to keep all of our pages that are using this pattern the same.

These states will typically be the only content on the page, and they provide a clear description, with optional actions to take when the user
reaches the state.

> The page-state component has built-in margins, and support for mobile layouts.

## Slots

| Name               | Values
| :------------------| :-----------------
| graphic            | The element to display as the graphic. Note: by default 100% `height` and `width` is set on the elements that are projected into this slot, but can be overridden if necessary.
| title              | The element containing the title text.
| message            | The element containing the message text.
| action             | The action button element location.

</PageSection>

<PageSection>

## CSS parts

| Name                        | Description
| :---------------------------| :----------------
| `root`                      | The root element.
| `graphic-container`         | The container element around the `graphic` slot.
| `title-container`           | The container element around the `title` slot.
| `message-container`         | The container element around the `message` slot.
| `actions-container`         | The container element around the `action` slot.


</PageSection>

<PageSection>

## CSS custom properties

| Name                                      | Description
| :-----------------------------------------| :----------------
| `--forge-page-state-graphic-height`         | Controls the height of the graphic in a desktop viewport. Default is `296px`.
| `--forge-page-state-mobile-graphic-height`  | Controls the height of the graphic in a mobile viewport. Default is `auto`.
| `--forge-page-state-width`                  | Controls the width of component in a desktop viewport. Default is `576px`.
| `--forge-page-state-mobile-width`           | Controls the width of component in a mobile viewport. Default is `70%`.
| `--forge-page-state-vertical-margin`        | Controls the vertical margin spacing between the elements. Default is `56px`.
