## Usage

For convenience Forge provides the `<forge-app-bar-menu-button>` component. This component uses an icon-button
with a predefined icon. The power of this component is that it will ensure a consistent experience for users
across all applications that consume it.

> It is expected that you place this component in the `start` slot of the app-bar.

</PageSection>
