# Divider

Use the divider component when you need to separate groups of content within a section.

</PageSection>

---

<PageSection>

## Properties/Attributes

<PropertyDef name="vertical" prop={false} type="boolean" defaultValue="null">

Causes the component to render vertically.

> **Note:** this can be used as an "existence" **attribute** only.

</PropertyDef>

</PageSection>

<PageSection>

## CSS parts

| Name                 | Description
| :--------------------| :---------------
| `root`               | The root element.

</PageSection>

<PageSection>

## CSS custom properties

| Name                                       | Description
| :------------------------------------------| :---------------
| `--forge-theme-border-color`               | Controls the color of the divider line.
| `--forge-divider-width`                    | Sets the stroke width of the divider line. Default is `1px`.
| `--forge-divider-margin`                   | Controls the `margin` style that is applied to the internal root element.

</PageSection>

<PageSection>

## Accessibility

- The `<forge-divider>` component automatically provides a role="separator" attribute.
  - If you would like to remove the divider from the accessibility tree, add `aria-hidden="true"` to the element.

</PageSection>
