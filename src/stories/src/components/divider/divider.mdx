## Usage

Use the divider component when you need to separate groups of content within a section. This is a simple component that was purely created to
ensure consistency.

This component will also properly inherit theme adjustments via CSS custom properties.

</PageSection>

---

<PageSection>

## Example

Using this component is very simple, use in place of the `<hr>` element like the following:

<LiveDemo 
fullWidth={true} 
codeHtml={DividerDefaultCodeHtml()}>
  <DividerDemo/>
</LiveDemo>

</PageSection>

## Properties/Attributes

<PropertyDef name="vertical" prop={false} type="boolean" defaultValue="null">

Causes the component to render vertically.

> **Note:** this can be used as an "existence" **attribute** only.

</PropertyDef>

<PageSection>

## CSS custom properties

| Name                                       | Description
| :------------------------------------------| :---------------
| `--forge-theme-border-color`               | Controls the color of the divider line.
| `--forge-divider-width`                    | Sets the stroke width of the divider line. Default is `1px`.
| `--forge-divider-margin`                   | Controls the `margin` style that is applied to the internal root element.

</PageSection>
