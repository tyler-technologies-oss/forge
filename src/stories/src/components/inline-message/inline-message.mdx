# Inline message

The inline-message component provides a static, targeted message to the use. It has a moderate amount of emphasis but should not be used for page specific messaging.
If you need to show a page or app wide messages, you should consider using the [banner component](?path=/story/components-banner--default) instead.

This component supports built-in predefined themes.

</PageSection>

## Properties/Attributes

<PropertyDef name="theme" type="string" prop={false} defaultValue="&quot;info-primary&quot;">

Allows for easily setting the theme color to use. Valid values: `danger`, `warning`, `success`, `info-primary`, `info-secondary`.

</PropertyDef>

<PageSection>

## Slots

| Name                           | Description
| :------------------------------| :----------------
| `default`                      | The default (unnamed) slot will render its content as the main text body.
| `icon`                         | The icon slot will render its content as the leading icon.
| `title`                        | The title slot will render its content above the main text body.

</PageSection>

<PageSection>

## CSS parts

| Name                       | Description
| :--------------------------| :----------------
| `root`                     | The root element.
| `container`                | The layout container element.

</PageSection>

<PageSection>

## CSS custom properties

| Name                                                  | Description
| :-----------------------------------------------------| :----------------
| `--forge-inline-message-theme-background`               | Sets the inline message's background color.
| `--forge-inline-message-theme-on-background`            | Sets the inline message's font color.
| `--forge-inline-message-theme-icon`                     | Sets the inline message's icon color.
| `--forge-inline-message-border-radius`                  | Sets the inline message's border radius.

</PageSection>


<PageSection>

## Accessibility

- If the inline-message is used to display an important notification to a user, use the `role="alert"` attribute on the `<forge-inline-message>` element.
  - This is equivalent to using `aria-live="assertive"`.
  - If the message is less urgent, you can use `aria-live="polite"` to wait until the user is finished with their current task.

</PageSection>
