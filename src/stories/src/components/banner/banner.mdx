# Banner

The banner component is intended to be used as a way to deliver a short but important message to the user. It has a high degree of emphasis and should not be used for general messaging.
If you need to show less important messages, or messages scoped to a more specific section of your page, you should consider using the [inline message component](?path=/story/components-inline-message--default). instead.

This component supports built-in predefined themes. It can be configured to be either static or dismissable. It can contain a button to trigger whatever action you need.

## Properties/Attributes

<PropertyDef name="dismissed" type="boolean" defaultValue="false">

Sets the banner to either the dismissed state or the undismissed state.

</PropertyDef>

<PropertyDef name="persistent" type="boolean" defaultValue="false">

Controls the visibility of the dismiss button.

</PropertyDef>

<PropertyDef name="theme" type="string" prop={false} defaultValue="&quot;&quot; (none)">

Allows for easily setting the theme color to use. Valid values: `danger`, `warning`, `success`, `info-primary`, `info-secondary`.
Note: Not assigning one of the mentioned themes will set the banner to use it's default colors.

</PropertyDef>

## Events

| Name                               | Description
| :----------------------------------| :---------------------
| `forge-banner-dismissed`             | Emits when the banner is dismissed.
| `forge-banner-undismissed`           | Emits when the banner is undismissed (shown again after being dismissed).

</PageSection>

<PageSection>

## Slots

| Name                           | Description
| :------------------------------| :----------------
| `default`                      | The default (unnamed) slot will render its content as the main text body.
| `icon`                         | The icon slot will render its content as the leading icon.
| `button`                       | The button slot will render its content after the main text body.

</PageSection>

<PageSection>

## CSS parts

| Name                  | Description
| :---------------------| :----------------
| `root`                | The root element.
| `container`           | The container layout element.
| `wrapper`             | The element used to control layout within the container.
| `icon-container`      | The element around the icon slot.
| `subsection-wrapper`  | The wrapper around the subsection text.
| `subsection-text`     | The element containing the text for the default slot.
| `action-subsection`   | The element containing the action for the button slot.
| `dismiss-icon-button` | The `<forge-icon-button>` element for the dismiss action.
| `dismiss-button`      | The `<button>` element within the `<forge-icon-button>` dismiss part.
| `dismiss-button-icon` | The `<forge-icon>` element for the dismiss action.

</PageSection>

<PageSection>

## CSS custom properties

| Name                                         | Description
| :--------------------------------------------| :----------------
| `--forge-banner-theme-background`              | Sets the banner's background color.
| `--forge-banner-theme-on-background`           | Sets the banner's font color.
| `--forge-banner-theme-icon`                    | Sets the banner's icon color.

</PageSection>

<PageSection>

## Accessibility

- If the banner is used to display an important notification to a user, use the `role="alert"` attribute on the `<forge-banner>` element.
  - This is equivalent to using `aria-live="assertive"`.
  - If the banner is less urgent, you can use `aria-live="polite"` to wait until the user is finished with their current task.
